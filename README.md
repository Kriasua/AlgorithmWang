# Algorithm Learning Project (AlgorithmProSln)

这是一个用于系统学习和实践数据结构与算法的 C++ 项目。项目涵盖了从基础语法、数据结构到高级算法的各类练习和实现。

## 🚀 项目结构

项目代码主要集中在 `AlgorithmPro/src/` 目录下，按类别进行组织：

### 1. 核心算法 (Algorithm Files)
包含了大量的算法专题练习，主要分类如下：

*   **基础算法**: 排序（QuickSort, sort.cpp）、位运算、随机化处理等。
*   **动态规划 (DP)**: 系统化的 DP 练习，涵盖一维、二维、三维 DP，背包问题（01、完全、多重、分组、有依赖），区间 DP，树形 DP，状压 DP 等。
*   **图论**: Dijkstra 算法、BFS/DFS、并查集 (UnionSet)、FloodFill 等。
*   **搜索与数据结构**: 
    *   二分搜索 (BinarySearch)
    *   滑动窗口 (SlideWindow)
    *   双指针 (DoublePtr)
    *   单调栈/单调队列 (MonotonicStack/Queue)
    *   前缀和/差分 (Difference)
    *   哈希表 (HashMap)
    *   前缀树 (TrieTree)
*   **专题练习**: LeetCode Hot 100、面试高频题 (Interview)、二叉树高频题 (TreeHighFrequencyQuestions) 等。

### 2. 基础与语法 (Grammar Study & Others)
*   `grammar study files/`: C++ 基础语法练习（如 `MyString.cpp`）。
*   `other files/`: 项目辅助文件，如预编译头文件 `pch.cpp`。

### 3. 头文件 (Header Files)
*   自定义数据结构实现，如 `Bitset.h`, `Heap.h` 等。

## 🛠️ 技术栈
*   **语言**: C++
*   **工具**: Visual Studio (vcxproj) / MSVC
*   **环境**: Windows
